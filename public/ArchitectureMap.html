4. Core Logic & Math Formulas

Net Asset Value (NAV)

NAV = (Total Premiums Paid) + (Dividend Share) + (Chit Share)

Dividend Share

Share = (Sum of all interestPaid globally) / (Total Members)

Chit Savings

Share = (Sum of all Expenditures where isChitPayment=true) / (Total Members)

Cash In Hand

Total Cash = Inflow(Premiums+Repaid) - Outflow(Loans+Exp+Chits)

5. Beginner's Fix Guide

  • UI/Visual Changes: Edit the files ending in page.tsx inside src/app/. Change the Tailwind classes for colors/spacing.
  • Math/Logic Changes: Edit the backend API files in src/app/api/.../route.ts.
  • PDF Report Formatting: Edit src/lib/pdf-reports.ts which uses jsPDF.
  • Database Updates: Edit prisma/schema.prisma and then run npx prisma db push.
